Debug Story Queue Processing

Open the browser console (F12) and navigate to the Story Generator app at localhost:3000

Steps to debug:

  1. Open browser console (F12)
  2. Go to Settings tab and check if there are any model configurations
  3. Go to Story Generator tab
  4. Enter a simple prompt like "A detective story"
  5. Click "Generate Story"
  6. Watch the console for debug messages
